Git

  • git init - инициализирует репозиторий.
  • git clone - клонирует удаленный репозиторий с GitHub себе на компьютер.
  • git add index.html - добавить файл в отслеживаемые файлы.
  • git status - проверить статус файла.
  • git commit -m "repo commit" - операция записи изменений в репозиторий Git.
  • git push - отправить изменения на удаленный репозиторий.
  • git pull - получить изменения c удаленного репозитория.
